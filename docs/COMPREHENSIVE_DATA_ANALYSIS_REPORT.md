## Appendix: Analysis Methodology

### Data Sources

**Collections Analyzed:**
- `orders` (5,711,551 documents)
- `deliverytrips` (4,745,547 documents)
- `merchants` (26,181 documents)
- `branches` (13,074 documents)
- `addresses` (2,040,538 documents)

**Time Period:** January 1, 2023 - December 31, 2025 (3 full years)

**Analysis Scripts:**
1. `monthlySeasonalAnalysis.js` - Monthly and seasonal trend analysis
2. `comprehensiveGlobalAnalysis.js` - Platform-wide metrics and cohort analysis
3. `quarterlyComparison.js` - Quarterly YoY comparisons
4. `deepMerchantAnalysis.js` - Merchant-level performance analysis

### Key Metrics Defined

**Orders:** Total orders placed on platform (from `orders` collection)

**Delivery Trips:** Orders that resulted in a delivery attempt (from `deliverytrips` collection)

**Completion Rate:** `completed trips / total trips * 100`

**Cancellation Rate:** `canceled trips / total trips * 100`

**Orders-to-Trips Conversion:** `delivery trips / orders * 100`

**YoY Growth:** `(current year - previous year) / previous year * 100`

**Active Merchant:** Merchant with at least 1 order in the time period

**Productive Merchant:** Merchant with >1,000 orders per year

**Merchant Cohorts:**
- Explosive Growth: >100% YoY growth
- High Growth: 50-100% YoY growth
- Moderate Growth: 10-50% YoY growth
- Stable: 0-10% YoY growth
- Declining: <0% YoY growth
- New: First orders in current year
- Churned: Had orders in previous year but none in current year

### Analysis Limitations

1. **Revenue Data:** Order totals not consistently populated; revenue analysis limited
2. **Geographic Data:** Partial area/governorate data; branch-level analysis used instead
3. **Customer Data:** Customer-level analysis not included in this report (focus on merchant/platform performance)
4. **Competitive Data:** No external competitive intelligence included; analysis based solely on internal platform data
5. **Causation:** Report identifies correlations and patterns but root cause analysis requires additional qualitative research (merchant interviews, customer surveys, etc.)
