Authentication Integration Example

This page demonstrates how to integrate authentication into your dashboard pages.

Step 1: Include the Auth Script

Add this script tag to the head of your HTML file:

<script src="/auth-utils.js"></script>

Step 2: Add User Profile Container

Add this div where you want the user profile to appear (usually in the sidebar):

<div id="user-profile-container"></div>

Step 3: That's It!

The auth-utils.js script will automatically:

  • Check if the user is authenticated when the page loads
  • Redirect to login page if not authenticated
  • Display user profile with avatar, name, and email
  • Add a logout button

Advanced: Custom JavaScript Access

You can also access user information programmatically:

// Get current user info console.log(authManager.user); // { name: "John Doe", email: "john@armadadelivery.com", avatar: "..." } // Check if authenticated console.log(authManager.authenticated); // true or false // Logout programmatically authManager.logout(); // Get full user details const userInfo = await authManager.getUserInfo(); console.log(userInfo);
